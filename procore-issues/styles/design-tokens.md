# Design tokens

Global styles live in `app/globals.css` per the Next.js App Router convention.
This file documents the token system used throughout the prototype.

## Color
- Primary (actions/highlights only): #F46A21
- Background: #F7F8FA
- Card: #FFFFFF
- Primary text: #1F2937
- Secondary text: #6B7280
- Border: #E5E7EB
- Success: #15803D
- Warning: #B45309
- Danger: #B91C1C
- Info: #1D4ED8

## Status → color
Open → Warning (orange) · In Progress → Info (blue) · Resolved → Success (green) · Blocked → Danger (red)

## Priority → color
High → Danger · Medium → Warning · Low → Success
